@startuml
participant "Main Function" as Main
participant "calculate()" as Calc
participant "getInput()" as Input

Main -> Calc: calculate()を呼び出す
activate Calc

Calc -> Input: getInput()を呼び出す
activate Input
Input --> Calc: 入力データを返す
deactivate Input

Calc --> Main: 計算結果を返す
deactivate Calc

Main -> Main: 結果を出力
@enduml
